# 🌱 Vridhi Mitra – AI-Powered Personalized Learning Platform for Rural Students

Vridhi Mitra is a mission-driven AI platform designed to bridge the educational gap for students in rural areas. By leveraging the Google Gemini AI engine, it provides high-quality, personalized learning guidance, automated assessments, and placement preparation that were previously inaccessible due to resource constraints.

## ✨ Key Features

### 🏢 Super Admin (Regional Oversight)
- 🌍 **Multi-Tenant Architecture** – Manage learning networks across different rural zones/colleges.
- 📊 **Impact Analytics** – System-wide tracking of student progress and educational growth indicators.

### 🛡️ Admin (Center Management)
- 👥 **Account Management** – Oversee student and faculty enrollment for local rural centers.
- 📚 **Academic Setup** – Curate courses and assign resources specifically for rural learner needs.
- 🎨 **Unified Design** – Clean, accessible "Deep Royal Purple" interface optimized for low-bandwidth and mobile access.

### 👨‍🏫 Faculty (Intelligent Instruction)
- 🤖 **AI Quiz Generator** – Automated quiz creation for **any subject** using Google Gemini (even without a pre-built question bank).
- 📂 **Digital Content Material (DCM)** – Upload and manage localized study resources (PDFs, Videos, Links).
- 📝 **Standardized Marks Entry** – Dedicated support for **FIAT**, **SIAT**, and **Model Exams** (100-mark scaling).

### 🎓 Student (Personalized Growth)
- 🤖 **Pragati AI Bot** – 24/7 intelligent learning assistant for concept clearing and study plans.
- 🎯 **AI Learning Paths** – Personalized roadmaps generated based on individual student performance.
- 📊 **Dynamic Dashboard** – Real-time progress charts and glassmorphic UI.
- 🎤 **AI Mock Interviews** – Career guidance and placement preparation with real-time AI feedback.

## 🚀 Quick Start & Installation

For a detailed, step-by-step setup guide for professors and developers, please refer to the:
👉 **[INSTALLATION_GUIDE.md](file:///c:/Users/Admin/Hackathon%20St.Joseph/INSTALLATION_GUIDE.md)**

### Basic Steps:
1.  **Clone:** `git clone https://github.com/kaaviyak2004/Vridhi-Mithra.git`
2.  **Database:** Create `vridhi_mitra` in MySQL.
3.  **Seed:** Run `node scripts/seedData.js` in the backend folder to generate the **5-college demo environment**.
4.  **Run:** Start both backend (`node server.js`) and frontend (`npm run dev`).
